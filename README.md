## ğŸ”¬ Formula Reference

### Holliday-Segar Method
- **0-10 kg:** 100 mL/kg/day
- **11-20 kg:** 1000 mL + 50 mL/kg for each kg above 10
- **>20 kg:** 1500 mL + 20 mL/kg for each kg above 20

### Rehydration
- **Standard:** 20 mL/kg
